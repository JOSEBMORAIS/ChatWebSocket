
# Chat WebSocket

## Como rodar o projeto:

1. Instale o Node.js se ainda não tiver.
2. Na pasta do projeto, rode:

   npm install ws

3. Inicie o servidor:

   node server.js

4. Acesse no navegador:

   http://localhost:3000

Pronto! Você verá a interface do chat em tempo real. Abra em outra aba ou navegador para testar com mais de um usuário.
